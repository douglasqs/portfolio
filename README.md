# Douglas Soares — Portfolio

Portfolio profissional trilíngue (português, inglês e espanhol), criado para apresentar experiência em desenvolvimento full-stack, inteligência artificial, computer vision e integrações de software com hardware.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm test
```

O deploy para GitHub Pages é executado automaticamente pelo workflow em `.github/workflows/deploy-pages.yml`.
